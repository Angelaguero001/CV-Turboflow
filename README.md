# CV-TurboFlow

Proyecto Escolar 
Aplicación web para gestión de currículums vitae con arquitectura TurboFlow. Permite administrar información personal, educación, experiencia laboral, habilidades, idiomas y hobbies.

## 🛠️ Tecnologías Utilizadas
- **Backend:** ASP.NET / .NET Framework
- **Base de Datos:** MySQL 8.0/9.0
- **Control de Versiones:** Git & GitHub

## 🗄️ Base de Datos
### Tablas incluidas:
- `educacion` - Formación académica
- `habilidades` - Competencias técnicas/blandas
- `trabajos` - Experiencia profesional
- `idiomas` - Idiomas y niveles
- `hobbies` - Intereses personales
